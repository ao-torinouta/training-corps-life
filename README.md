# 教育隊生活
教育隊の生活リズムをモチーフにした生活支援アプリです。

コンセプト

生活を管理するアプリではありません。

「今は何の時間か」を知らせ、スマホを置くためのアプリです。

ToDo管理ではなく、
生活の「現在課目」を表示し、自分で判断して行動することを目的としています。

スマホを見る時間は増え続けています。

教育隊では「今は何の時間か」が明確でした。

教育隊生活は、その考え方を日常生活へ取り入れるために制作した個人プロジェクトです。

⸻

主な機能

* 現在時刻表示
* 現在課目・次課目表示
* 課目開始・課目完了・10分延長
* 教官メッセージ表示
* ホーム画面追加対応（PWA）
